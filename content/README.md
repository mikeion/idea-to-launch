# Project Collaboration Space

Welcome to our shared project workspace! This Obsidian vault is organized to help us collaborate effectively on projects.

## Structure
- **Projects/**: Individual project documentation
- **Meetings/**: Meeting notes and summaries
- **Templates/**: Reusable templates for various content types
- **Resources.md**: Useful links and references
- **Kanban Board.md**: Current project status board
- **Project Parameters.md**: Guidelines for our projects

## Getting Started
1. Make sure you have [Obsidian](https://obsidian.md/) installed
2. Install the Kanban and Dataview plugins
3. Browse the Projects folder to see what we're currently working on
4. Check the Kanban Board for tasks that need attention

## Plugins We Use
- **Kanban**: For visual project management
- **Dataview**: For creating dynamic views of our content

## Contributing
- Use the templates in the Templates folder when creating new content
- Follow the project parameters when proposing new ideas
- Keep the Kanban board updated with your progress
